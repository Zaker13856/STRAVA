# ZAK / DZ Cycling AI Lab

Due video cinematici HTML per il progetto Cycling AI Lab.

## Contenuto

### `zak-cycling-lab/`
Video personale — storia running → ciclismo → dashboard. 12 scene, 80 secondi.

### `dz-cycling-lab/`
Video prodotto — pitch della dashboard DZ. 13 scene, 96 secondi.

### `debrief_cycling_danilo.pdf`
Debrief uscite Lago di Bracciano e Passoscuro–Manziana–Sasso (giugno 2026).

## Tecnologia
- HTML5 self-contained con engine scene custom (`requestAnimationFrame`)
- Audio: AC/DC Thunderstruck
- Video clip: 10s cycling
- Mobile responsive (portrait + landscape)
- PWA ready (manifest.json)
